  $ npm install

┌──────────────────────────────────────────────────────────────────────────────┐
│ API ENDPOINTS                                                                │
└──────────────────────────────────────────────────────────────────────────────┘

Blog Endpoints:
  GET /api/metadata.json        → Developer profile + API info
  GET /api/notes.json           → List of all notes
  GET /api/notes/[slug]         → Individual note with content

Aggregator Endpoint:
  GET /api/notes                → Aggregated notes from all sources

All endpoints:
  • Return valid JSON
  • Support CORS
  • Include caching headers
  • Handle errors gracefully
